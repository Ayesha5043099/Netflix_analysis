#  Netflix Content Analysis Pipeline

A complete data engineering & analytics project built using:

- **Python (Pandas)**
- **PostgreSQL**
- **pgAdmin**
- **SQL**

---

##  Project Overview

This project builds an end-to-end data pipeline:

1. **Extract** raw Netflix dataset (CSV)
2. **Transform** data using Python (cleaning & preprocessing)
3. **Load** into PostgreSQL (normalized relational schema)
4. **Analyze** data using SQL queries

---

## Tech Stack

- Python (Pandas, NumPy)
- PostgreSQL
- pgAdmin
- SQL
- psycopg2
